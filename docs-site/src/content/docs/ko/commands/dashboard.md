---
title: mf dashboard
description: 로컬 mustflow 대시보드를 위해 예약된 명령입니다.
---

`mf dashboard`는 로컬에서 mustflow 워크플로를 화면으로 확인하고 관리하는 대시보드 기능을 위해 예약된 명령입니다.

현재 이 기능은 아직 구현되지 않았습니다. 명령을 실행하면 미구현 안내 메시지를 출력하고 종료 코드 `1`로 종료합니다.

## 현재 동작

```sh
npx mf dashboard
```

이 명령은 서버를 띄우거나 파일을 수정하지 않습니다.

## 구조화된 출력

`mf dashboard`는 현재 JSON 출력을 지원하지 않습니다.

자동화 도구나 에이전트는 이 명령을 실제 작업 명령으로 간주하면 안 됩니다.

## 도움말 및 종료 코드

```sh
npx mf dashboard --help
```

- 종료 코드 `0`: 도움말을 출력했습니다.
- 종료 코드 `1`: 대시보드 기능이 아직 구현되지 않았습니다.
