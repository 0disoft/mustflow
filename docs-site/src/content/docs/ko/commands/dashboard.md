---
title: mf dashboard
description: 로컬 mustflow 대시보드용으로 예약된 명령입니다.
---

`mf dashboard`는 로컬에서 mustflow 문서 흐름을 시각적으로 확인하고 수정하는 대시보드 명령으로 예약되어 있습니다.

현재는 기능을 구현하지 않았습니다. 명령을 실행하면 구현 전이라는 안내만 출력하고 종료 코드 `1`로 끝납니다.

## 현재 동작

```sh
npx mf dashboard
```

이 명령은 서버를 띄우거나 파일을 수정하지 않습니다.

## 구조화된 출력

`mf dashboard`는 현재 JSON 출력 형식을 제공하지 않습니다.

자동화나 에이전트는 이 명령을 사용 가능한 작업 명령으로 취급하지 말아야 합니다.

## 도움말과 종료 코드

```sh
npx mf dashboard --help
```

- 종료 코드 `0`: 도움말을 출력했습니다.
- 종료 코드 `1`: 대시보드 기능이 아직 구현되지 않았습니다.
