---
title: 검토 중인 기능
description: 아직 구현하지 않은 mustflow 기능 후보입니다.
---

이 문서는 현재 동작을 설명하는 문서가 아닙니다. mustflow에서 검토 중인 후보를 기록해 두기 위한 메모입니다.

후보 기능은 실제 구현 전까지 기본 템플릿, 명령 계약, 검증 규칙의 일부로 보지 않습니다.

## 후보 목록

| 항목 | 상태 | 현재 결정 |
| --- | --- | --- |
| `mf dashboard` | 검토 중 | 명령 이름만 예약되어 있으며 아직 구현하지 않았습니다. |
| 커뮤니티 스킬 저장소 | 검토 중 | 외부 스킬 설치·갱신 규칙은 아직 정하지 않았습니다. |
| 스킬 팩 설치 | 검토 중 | 기본 스킬과 선택 스킬의 경계가 안정된 뒤 검토합니다. |
| `.mustflow/work-items/` | 검토 중 | 기본 템플릿에는 넣지 않고, 선택 기능 후보로만 남깁니다. |
| `mf orient` | 검토 중 | 현재는 `mf context`, `mf map`, `mf help` 조합으로 탐색합니다. |
| `mf refresh` | 검토 중 | 현재는 `mf update`와 `mf check --strict`로 지침 상태를 확인합니다. |
| 도구별 어댑터 | 검토 중 | 특정 에이전트 제품 이름을 기본 파일명이나 필수 규칙으로 만들지 않습니다. |

## 추가 전 조건

새 기능은 다음 조건을 만족할 때만 공개 동작으로 옮깁니다.

- `mf init`이 설치하는 기본 문서 흐름을 불필요하게 키우지 않아야 합니다.
- 에이전트가 추측 없이 실행할 수 있는 명령 계약이 있어야 합니다.
- `mf check --strict`나 별도 검사로 잘못된 사용을 확인할 수 있어야 합니다.
- 사람이 직접 읽고 수정해도 부담이 없어야 합니다.
