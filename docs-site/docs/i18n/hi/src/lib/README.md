यह डायरेक्टरी छोटे जनरेशन सहायक रखती है जिन्हें docs-site के कई रूट साझा करते
हैं।

- `machine-readable.mjs`: `ai.txt`, `llms.txt`, `llms-full.txt` और
  `robots.txt` प्रतिक्रियाएँ बनाता है।

रूट फ़ाइलों को छोटा रखें। सार्वजनिक मेटाडेटा टेक्स्ट के स्रोत मान
`../config/machine-readable.mjs` में प्रबंधित करें।
