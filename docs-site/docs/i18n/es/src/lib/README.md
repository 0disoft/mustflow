Idiomas: [Inglés](../../../../../src/lib/README.md) · [Coreano](../../../ko/src/lib/README.md) · [Chino](../../../zh/src/lib/README.md) · [Español](README.md) · [Francés](../../../fr/src/lib/README.md) · [Hindi](../../../hi/src/lib/README.md)

Este directorio contiene pequeños ayudantes de generación compartidos por varias
rutas de docs-site.

- `machine-readable.mjs`: Genera respuestas para `ai.txt`, `llms.txt`,
  `llms-full.txt` y `robots.txt`.

Mantén los archivos de ruta simples. Gestiona los valores fuente del texto de
metadatos públicos en `../config/machine-readable.mjs`.
