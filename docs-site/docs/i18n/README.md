# docs-site README Translations

The English README files in `docs-site/` and `docs-site/src/**/README.md` are
the source documents for docs-site contributor notes. Keep the localized copies
in this directory aligned when source structure, commands, configuration files,
or style ownership changes.

## Maintenance Notes

- Keep the language switcher line in every source and localized README in sync.
- Update the English source first, then update each localized copy.
- Keep commands, file names, and relative paths identical unless a locale note
  needs a different sentence around the same literal value.
