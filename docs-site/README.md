# mustflow docs site

mustflow.github.io에 배포할 소비자용 문서 사이트입니다.

이 사이트는 `mf init`으로 사용자 저장소에 복사되는 문서가 아닙니다. mustflow가 만드는 파일과 설정의 의미를 설명하는 참고 문서입니다.

## 명령어

```sh
bun run dev
bun run check
bun run build
bun run preview
```

루트에서는 다음 명령을 사용할 수 있습니다.

```sh
bun run docs:dev
bun run docs:check
bun run docs:build
bun run docs:preview
```
